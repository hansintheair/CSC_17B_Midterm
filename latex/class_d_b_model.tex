\subsubsection*{template$<$class T$>$\newline
class DBModel$<$ T $>$}
\label{class_d_b_model_DBModel}%
\Hypertarget{class_d_b_model_DBModel}%
 This class provides the interface for database IO. It is implementad as a template class, and can store records of either the \mbox{\hyperlink{class_catalog}{Catalog}} or \mbox{\hyperlink{class_account}{Account}} classes. The acceptable datatypes are limited to any data type that defines a display method.
