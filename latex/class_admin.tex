\doxysubsection{Detailed Description}
\label{class_admin_Admin}%
\Hypertarget{class_admin_Admin}%
 This class serves as the \mbox{\hyperlink{class_admin}{Admin}} controller. It contains the nescessary functions to fascilitate the admin user role interactions.

\mbox{\hyperlink{class_admin}{Admin}} accounts have privilege to view/edit another \mbox{\hyperlink{class_account}{Account}}, view store earnings, or modify the store \mbox{\hyperlink{class_catalog}{Catalog}}.

\mbox{\hyperlink{class_admin}{Admin}} accounts are created or logged in from the \mbox{\hyperlink{class_home}{Home}} class main menu.

There is one primary admin that cannot be deleted, called admin.

\mbox{\hyperlink{class_admin}{Admin}} record attributes are stored in \mbox{\hyperlink{class_account}{Account}} objects.
