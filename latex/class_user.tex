\doxysubsection{Detailed Description}
\label{class_user_User}%
\Hypertarget{class_user_User}%
 This class serves as the \mbox{\hyperlink{class_user}{User}} controller. It contains the nescessary functions to fascilitate the user role interactions.

\mbox{\hyperlink{class_user}{User}} accounts can shop the \mbox{\hyperlink{class_catalog}{Catalog}} and place orders.

\mbox{\hyperlink{class_user}{User}} accounts are created or logged in from the \mbox{\hyperlink{class_home}{Home}} class main menu.

\mbox{\hyperlink{class_user}{User}} record attributes are stored in \mbox{\hyperlink{class_account}{Account}} objects.
