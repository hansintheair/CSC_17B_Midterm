\doxysubsection{Detailed Description}
\label{class_home_Home}%
\Hypertarget{class_home_Home}%
 This class serves as the main entry point into the shopping store application. The {\ttfamily \mbox{\hyperlink{class_home_af37e6f3e876567384e5bfe8a425cd7cd}{Home\+::main()}}} method launches the main menu from which all other parts of the application are accessible to the user. Users and admins use this page to log in.

\mbox{\hyperlink{class_user}{User}} accounts\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_user}{User}} accounts can shop the catalog and place orders.
\end{DoxyItemize}

\mbox{\hyperlink{class_admin}{Admin}} accounts\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_admin}{Admin}} accounts have the privilege to view/edit other accounts, view store earnings, or modify the store catalog.
\item There is one primary admin account that cannot be deleted. ~\newline
 Username\+: admin ~\newline
 Password\+: admin ~\newline
 
\end{DoxyItemize}
