\doxysubsection{Detailed Description}
\label{class_account_Account}%
\Hypertarget{class_account_Account}%
 This class stores \mbox{\hyperlink{class_user}{User}} and \mbox{\hyperlink{class_admin}{Admin}} account data and provides accessors for reading and manipulating user/admin attributes.

\mbox{\hyperlink{class_d_b_model}{DBModel}} is used to store records of \mbox{\hyperlink{class_account}{Account}} users in a binary database file.
