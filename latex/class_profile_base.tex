\doxysubsection{Detailed Description}
\label{class_profile_base_ProfileBase}%
\Hypertarget{class_profile_base_ProfileBase}%
 This class serves as the Baseclass for both the \mbox{\hyperlink{class_user}{User}} and \mbox{\hyperlink{class_admin}{Admin}} controllers. It contains the shared functions to fascilitate the user and admin role interactions.

\mbox{\hyperlink{class_profile_base}{Profile\+Base}} record attributes are stored in \mbox{\hyperlink{class_account}{Account}} objects.
